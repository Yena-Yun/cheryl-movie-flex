## 🧨 프로젝트 소개

네이버 API를 사용하여 영화를 검색하고 즐겨찾기할 수 있는 모바일 SPA 구성

## ✈ 배포 링크

##

## ✨ 구현 기능

1. 네이버 API로 영화 정보 받아오기
2. Search 탭에서 영화 검색 및 즐겨찾기
3. Favorites 탭에서 즐겨찾기한 영화 확인
